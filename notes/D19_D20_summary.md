# D19-D20：自动筛选最优参数与优化前后对比

## 1. 任务目标

本阶段基于前期 Zemax 参数扫描结果，对不同参数组合下的成像评价指标进行自动筛选，并整理优化前后对比结果。

D19 的目标是根据扫描结果 CSV 自动计算或读取评价指标，并筛选出综合评分最高的参数组合。  
D20 的目标是将初始设计与最优设计进行对比，整理 MTF、Spot 或评分等结果，为后续项目报告和简历展示准备素材。

## 2. 输入文件

本阶段主要输入文件包括：

- `sweep_results.csv`：参数扫描结果表，包含参数值、MTF、RMS Spot、score 等指标；
- 初始 Zemax 模型文件：用于作为 before 对比对象；
- 最优参数信息：由 D19 自动筛选生成。

## 3. D19：自动筛选最优参数

D19 中使用 Python 读取 `sweep_results.csv`，并根据设定的评分指标筛选最优参数。当前评分逻辑主要参考 MTF 和 RMS Spot，其中 MTF 越高代表成像细节传递能力越好，RMS Spot 越小通常代表像点越集中。

基本流程为：

1. 读取参数扫描结果 CSV；
2. 检查每一组参数对应的 MTF、RMS Spot 和 score；
3. 根据 score 选择最优参数组合；
4. 将最优参数保存为 `best_design.json` 或 `best_design.txt`；
5. 为后续优化前后对比提供输入。

## 4. D20：优化前后对比

D20 中将初始设计与 D19 筛选出的最优设计进行对比，主要关注优化前后 MTF、Spot 或综合评分是否发生改善。

基本流程为：

1. 读取初始设计结果；
2. 读取最优设计结果；
3. 对比关键评价指标；
4. 整理优化前后对比图；
5. 将结果保存到 `before_after` 或对应结果文件夹中。

## 5. 当前产出

本阶段主要产出包括：

- `sweep_results.csv`
- `best_design.json` 或 `best_design.txt`
- `mtf_vs_thickness.png`
- `score_vs_thickness.png`
- `before_after_mtf.png`
- D19-D20 阶段说明文档

## 6. 当前理解

D19-D20 的核心意义是把参数扫描结果从“人工观察”变成“自动筛选”。  
通过引入评分函数，可以让程序自动从多组参数中选出相对更优的结构，而不是手动逐个查看结果图。

D20 的优化前后对比则是项目展示中非常重要的一步。只有把 before 和 after 放在一起比较，才能说明自动化扫描和筛选确实带来了性能变化。

## 7. 当前局限

当前阶段的结果依赖于前期参数扫描数据。如果 MTF、RMS Spot 或 score 的提取还不够完善，那么最优参数的判断也会受到影响。

后续需要继续完善真实 Zemax 分析结果的自动导出，并保证所有对比都在相同视场、波长、空间频率和评价指标下进行。

## 8. 后续计划

下一步需要将 D19-D20 的最优参数筛选逻辑整合进第 4 周的 workflow 中，使其可以在一键运行流程中自动完成：

配置读取 → 参数扫描 → 保存 CSV → 筛选最优参数 → 生成对比图 → 生成报告。
